# My Countries App

My Countries App is an Android application built with Jetpack Compose that allows users to browse
and search for information about countries.

## Demo

![App Demo](https://your-gif-url.com/demo.gif)

## Features

* **Browse Countries:** View a list of countries.
* **Search:** Search for specific countries.
* **Country Details:**  View detailed information about a selected country.

## Technologies Used

* **UI:** Jetpack Compose
* **Architecture:** MVVM (Model-View-ViewModel)
* **Asynchronous Programming:** Kotlin Coroutines
* **Dependency Injection:** Koin
* **Networking:** Apollo & OkHttp
* **Image Loading:** Coil
* Etc.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Android Studio Iguana | 2023.2.1 or later
* Android SDK 34

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/MyCountriesApp.git
   ```
2. Open the project in Android Studio.
3. Let Android Studio download the dependencies and sync the project.
4. Run the app on an emulator or a physical device.
